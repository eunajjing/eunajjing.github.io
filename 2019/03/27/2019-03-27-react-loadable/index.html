react-loadable

3월 27, 2019 개발공부, React

react-loadable

yarn add react-loadable

장점

  • 페이지 로딩을 할 때부터 청크파일들을 다른 자바스크립트파일들과 동일한 시점에서 로딩을 시작 할 수 있다.

  • 서버사이드 렌더링과 함께 쉽게 코드 스플리팅을 할 수 있다.

  • 로딩 중일 때 렌더링할 컴포넌트를 따로 지정할 수 있다.

  • Preloading

    • 렌더링되지 않아도 특정 함수를 호출하여 미리 불러온다.
    • 링크 호버 시에 로딩을 시작한다

  • 서버사이드렌더링과 함께 쓰는 방법
