깃허브를 이용해 html, css, js 파일 호스팅

완성된 파일에 yarn build 명령어를 이용해 최적화한다.

package.json 파일에 hompage라는 필드를 만들어준다.

gh-pages 플러그인을 설치한다.

yarn add gh-pages -D

package.json의 scripts 필드에 필드를 추가한다.

cmd에 yarn run deploy 명령어 실행
