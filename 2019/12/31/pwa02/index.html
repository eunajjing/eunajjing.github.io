특정 모듈의 설치

이번 PWA는 React를 가지고 실습 했습니다. CRA를 이용하지 않고 한땀한땀 쳐내려갔기 때문에, 일단 React 관련 모듈부터 설치해줬습니다.

이후 Webpack과 babel 관련 모듈을 설치합니다. 해당 모듈은 개발 시에만 필요한 것이므로, devDependencies로 둡니다.
